
# Excel-Data-Manipulation

Small, simple Python script for basic Excel data tasks (cleaning, filtering, small transforms).

Requirements
- Python 3.8+
- pandas
- openpyxl

Quick install
```sh
python3 -m venv .venv
source .venv/bin/activate
pip install pandas openpyxl
```

Quick run
```sh
python app.py input.xlsx output.xlsx
```
